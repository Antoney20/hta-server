# Security Policy

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.** This exposes the problem to everyone before it can be fixed.

Report security issues privately by emailing:

**📧 hbtap@uonbi.ac.ke**

Please include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix, if available

We will acknowledge your report within **5 working days** and keep you informed as we investigate and resolve it.

---

## Responsible Disclosure

We ask that you:

- Allow us reasonable time to patch before any public disclosure
- Avoid accessing or modifying data that is not yours
- Act in good faith toward users and Kenya's health systems

We appreciate responsible engineer who help keep BPTAP secure.
